# economics-of-slavery

A brief cliometrics analysis of US slavery
This project consists of two complementary analyses:

1. Cliometric Analysis (R)
Uses 1790 and 1860 U.S. Census data to examine changes in slavery intensity over time.

Dependent variable: slaves per capita
Independent variable: time (year indicator)
Includes state-level controls and visual analysis
2. Economic Analysis (Python)
Examines the relationship between slavery and economic development.

Dependent variable: manufacturing capital per capita
Independent variable: slavery (slaves per capita)
Estimated using OLS regression
Together, these approaches provide both historical context and economic interpretation.
##Limitations
Due to the results of this study, it is best not to suggest a causal relationship between the variables. In addition to this due to the sample size being low (n < 50)  there is more of a correlation than causation. 

For further results on this you may see the paper of this study on Github.
